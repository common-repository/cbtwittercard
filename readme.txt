Full featured Twitter Card For Wordpress

== Description ==

Twitter cards is an awesome plugin which makes it possible for you to attach media experiences to Tweets that link to your content. 
Simply add this plugin in wordpress and add additional info in your page. Users who Tweet links of your content will have a “card” added to the Tweet that’s visible to all of their followers.

Learn more http://wpboxr.com/product/cbx-twitter-card

Features of the plugin include:

 * Enable Most Twitter Card Types for WordPress
 * Display Media Experience to Tweets to Your Content
 * Great for Ecommerce Information
 * Default Post Type Support
 * Works with any Custom WP Template

== Installation ==

1. Upload `cbtwittercard.zip` to the `/wp-content/plugins/` directory or search in wordpress dir from wordpress for "CBX Twitter Card For Wordpress"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You'll now see a menu called "Twitter Card" in left menu of word, start from there
